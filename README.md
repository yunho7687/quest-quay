# quest-quay
Step 0: install python 3.4+
### Sept 1: create a Python venv     
```bash
python -m venv venv
```

### Sept 2: install the requirements      
```bash
pip install -r requirements.txt
```

### Sept 3: 
```bash
flask run
```
Open a browser and go to URL:   
`http://<IP>:5000 `    
   
`http://127.0.0.1:5000 `
